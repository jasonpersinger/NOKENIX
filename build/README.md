# Build Layout

This directory contains the NOKENIX live-build configuration and the thin project wrappers around it.

## Structure

- `auto/`: live-build automation entrypoints
- `config/`: live-build configuration tree

## Important Rule

Only commit the human-authored parts of `config/`, such as:

- `config/package-lists/`
- `config/includes.chroot/`
- `config/includes.binary/`
- `config/hooks/`

Do not commit generated `config/*` files containing `LB_*` values. Those are regenerated by `lb config` and are ignored by `.gitignore`.
